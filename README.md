# Mushaf Al Noor Online Quran Academy

Mushaf Al Noor Online Quran Academy is a professional online Quran learning platform for kids and adults worldwide. The project combines modern React and TypeScript architecture with a polished educational experience.

## Project overview

- Modern Vite + React + TypeScript frontend
- Tailwind CSS and shadcn/ui design system
- Public-facing course pages and student/admin experience
- SEO-ready metadata and branding aligned to Mushaf Al Noor

## Development

```sh
npm install
npm run dev
```

## Production build

```sh
npm run build
```

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- React Router
- TanStack Query
